# Example PeakGuideR peak matrix

Small example peak matrix object used to demonstrate the PeakGuideR
workflow.

## Usage

``` r
data(example_pkm)
```

## Format

A list containing at least:

- mass:

  m/z values of detected features.

- intensity:

  Pixel-by-feature intensity matrix.

- pos:

  Pixel coordinates.

## Source

Internal example dataset.

## Details

This dataset is intended for examples, tests and vignette execution. It
is a reduced peak matrix and should not be used for biological
interpretation.
